Luas Segitiga

Rumus Luas Segitiga:

L = ½ x Alas x Tinggi

Keterangan :
L = Luas Segitiga
Alas = Alas Segitiga
Tinggi = Tinggi Segitiga

Keliling Segitiga

Rumus Keliling Segitiga:

K = Sisi A + Sisi B + Sisi C

Keterangan :
K = Keliling Segitiga
Sisi A = Sisi Pertama Segitiga
Sisi B = Sisi Kedua Segitiga
Sisi C = Sisi Ketiga Segitiga
